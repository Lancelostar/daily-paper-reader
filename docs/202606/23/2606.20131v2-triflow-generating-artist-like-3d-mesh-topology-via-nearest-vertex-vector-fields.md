## 摘要
我们提出TriFlow，一种新的生成方法，直接从输入几何条件（如符号距离场）生成具有艺术家风格三角形拓扑的紧凑3D网格。我们的关键洞察是将网格拓扑表示为定义在曲面上的最近顶点向量场（NVF），其中每个点编码其在局部重心坐标系中与最近三角形顶点的关联。我们训练一个潜流匹配模型来合成该场，从而能够根据输入几何条件生成拓扑。为了提取连贯的网格，我们使用生成的NVF对表面区域进行聚类，并通过拓扑感知优化指导约束二次误差度量（QEM）网格简化。这产生了与输入几何紧密匹配的输出网格，同时展现出结构化的、类似艺术家的连接。实验表明，与最先进的基于学习的方法相比，TriFlow实现了更强的泛化能力和显著改善的拓扑质量，同时Chamfer距离降低了90%，速度提升了8倍。

## Abstract
We present TriFlow, a new generative approach for producing compact 3D meshes with artist-like triangle topology directly from input geometry conditions such as signed distance fields. Our key insight is to represent mesh topology as a nearest-vertex vector field (NVF) defined over the surface, where each point encodes its association to the nearest triangle vertex in the local barycentric frame. We train a latent flow-matching model to synthesize this field, enabling topology generation conditioned on the input geometry. To extract a coherent mesh, we cluster surface regions using the generated NVF and guide a constrained quadric error metric (QEM) mesh simplification with topology-aware optimization. This yields output meshes that closely match the input geometry while exhibiting structured, artist-like connectivity. Experiments demonstrate that TriFlow achieves stronger generalization and significantly improved topology quality compared to state-of-the-art learning-based approaches, alongside 90% lower Chamfer Distance and an 8x speedup.

- **研究动机**：现有的 3D 生成模型（如基于隐式场的方法）虽然能产生高保真几何，但通过 Marching Cubes 提取的网格通常高度过密、拓扑杂乱，不适合下游的编辑、动画和实时渲染。另一方面，直接生成离散网格序列的自回归方法（如 MeshGPT、TreeMeshGPT）虽能产生艺术家风格的拓扑，但存在逐 token 推理慢、误差累积、泛化能力差等问题。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将网格拓扑建模为定义在曲面上的**最近顶点向量场（Nearest-Vertex Vector Field, NVF）**，从而将离散的拓扑预测转化为连续场建模，避免自回归序列建模的缺陷。
- **技术细节**：
  - **NVF 定义**：对曲面上任意点 \(p\)，NVF 取值 \(t(p) = v_n - p\)，其中 \(v_n\) 是包含 \(p\) 的三角形中重心坐标权重最大的顶点。该场自然地将每个三角形划分为三个区域，区域连通性对应顶点连通性，且场是分段连续的。
  - **体素化**：将 NVF 和输入 SDF 稀疏体素化为 512³ 网格，并训练自编码器得到 64³ 的潜码。
  - **潜流匹配模型**：基于 TRELLIS 架构，采用条件流匹配（conditional flow matching）训练，以输入 SDF 潜码和用户控制的拓扑参数（目标面数、四边形比例）为条件，生成 NVF 潜码，再解码得到体素化 NVF。
  - **网格提取**：
    1. 对预测的 NVF 进行双边滤波去噪。
    2. 利用稠密 Marching Cubes 网格作为代理网格，将 NVF 转移到其顶点。
    3. 使用**分水岭算法（watershed）** 根据 NVF 将表面顶点聚类到各目标顶点区域（基于预测位移大小初始化根节点，优先队列扩展）。
    4. 执行**约束 QEM 简化**：只允许同一聚类内的边折叠，并对根节点加入位置约束二次型，使收缩点靠近预测的目标位置，同时保留几何保真度。
  - **数据增强**：为增强对噪声和不规则几何的鲁棒性，训练时对网格施加随机 3D 空间变形场，扭曲几何和对应 NVF。
