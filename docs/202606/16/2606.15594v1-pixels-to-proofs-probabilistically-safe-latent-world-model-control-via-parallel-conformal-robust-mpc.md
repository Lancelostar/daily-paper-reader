## 摘要
我们提出SLS^2，一个利用鲁棒模型预测控制（MPC）在学习到的潜在世界模型中进行基于像素的安全反馈运动规划的框架。我们的方法训练了一个动作条件的联合嵌入世界模型，具有紧凑的马尔可夫潜在状态，能够通过学习的潜在动力学实现高效的基于梯度的轨迹优化。为了在不完美的潜在预测下确保真实系统的安全性，我们通过共形预测来校准潜在误差界限和鲁棒的潜在空间约束集，从而指导GPU加速的系统级综合（SLS）鲁棒MPC方案。我们还学习并共形化一个潜在约束检查器，使SLS规划器能够在闭环执行期间施加概率安全约束。我们在基于视觉的控制任务上评估了该方法，与潜在世界模型和安全规划基线相比，它在目标到达性能和安全方面均有改进。

## Abstract
We present SLS^2, a framework for safe feedback motion planning from pixels using robust model predictive control (MPC) in learned latent world models. Our approach trains an action-conditioned joint-embedding world model with compact Markovian latent states, enabling efficient gradient-based trajectory optimization through learned latent dynamics. To enforce safety for the true system despite imperfect latent predictions, we inform a GPU-accelerated system level synthesis (SLS) robust MPC scheme with conformal prediction to obtain calibrated latent error bounds and robust latent-space constraint sets. We further learn and conformalize a latent constraint checker, allowing the SLS planner to impose probabilistic safety constraints during closed-loop execution. We evaluate our method on vision-based control tasks, where it improves both goal-reaching performance and safety over latent world-model and safe-planning baselines.

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想
SLS^2由三部分组成：
1. **训练一个规划友好的潜在世界模型**：使用联合嵌入预测架构（JEPA），学习紧凑的马尔可夫潜在状态和可微分的潜在动力学，支持基于梯度的轨迹优化。
2. **共形校准不确定集合**：对潜在动力学预测误差和潜在约束分类器误差进行共形预测校准，得到具有概率覆盖保证的误差集合。
3. **鲁棒MPC规划**：将校准后的不确定集合集成到GPU加速的SLS框架中，优化名义轨迹和反馈控制器，同时确保可达管（reachable tube）始终位于安全集合和域内支持集合内。

### 2.2 关键技术细节

**A. 潜在世界模型训练（Sec. 4.1）**
- 编码器：ViT-Tiny，将RGB图像（224×224）映射为潜在嵌入 \(z_t \in \mathbb{R}^{n_z}\)。
- 马尔可夫潜在状态：\(s_t = [z_t, \Delta z_t, \ldots, \Delta^K z_t]\)，其中 \(\Delta z_t = z_t - z_{t-1}\)，K为马尔可夫阶数。
- 动力学预测器 \(f_\phi\)：MLP，输入当前潜在状态和动作，输出下一状态 \(\hat{s}_{t+1}\)。
- 训练损失：多步预测损失 \(L_{pred}\)（H步自回归均方误差）、SIGReg防止表示坍塌、时间直化损失（temporal straightening）鼓励速度向量对齐。
- 总损失：\(L = L_{pred} + \lambda_{sig} L_{sig} + \lambda_{str} L_{str}\)。

**B. 潜在约束分类器（Obstacle Classifier）**
- 使用冻结编码器将标注好的障碍/非障碍图像映射为潜在嵌入，训练一个小型MLP分类器 \(c_\psi\)，输出得分（正数表示安全，负数表示不安全）。
- 对不安全类别的校准数据集，计算非符合分数 \(r_i = \max(0, c_\psi(z_i))\)，取 \((1-\delta)\) 分位数作为阈值 \(\eta\)。规划时，若 \(c_\psi(z) < \eta\)，则判定不安全。

**C. 共形不确定量化（Sec. 4.2）**
- **动力学误差校准**：在保留的校准集上计算残差 \(e_i = s^+_i - f_\phi(s_i, u_i)\)，训练一个协方差模型 \(\Sigma_\psi(s,u)\)（状态-动作相关椭圆形状）。通过共形预测选择临界值 \(q\)，使未来一步误差满足 \(e^\top \Sigma_\psi^{-1} e \leq q\) 的概率至少为 \(1-\delta/T\)。
- **域内（In-Domain）支持校准**：拟合校准潜在状态的高斯椭球，以覆盖 \((1-\alpha_{ID})\) 的比例，得到域内集合 \(I = \{s \mid (s-\mu)^\top \Sigma_{ID}^{-1}(s-\mu) \leq q_{ID}\}\)。

**D. 鲁棒SLS-MPC规划（Sec. 4.3）**
- 将校准后的误差集合写为 \(E(s,u) = \sqrt{q} L_\psi(s,u) \mathcal{B}^{n_s}\)，其中 \(L_\psi\) 是 \(\Sigma_\psi\) 的Cholesky因子。
- 在名义轨迹 \((\zeta_k, v_k)\) 附近线性化动力学，得到 \(A_k, B_k\)，并设置 \(E_k = \sqrt{q} L_\psi(\zeta_k, v_k)\)。
- SLS优化变量：名义轨迹、控制以及响应矩阵 \(\Phi^s, \Phi^u\)，满足：
  \[
  \Phi^s_{k+1,j} = A_k \Phi^s_{k,j} + B_k \Phi^u_{k,j},\quad \Phi^s_{j+1,j} = E_j
  \]
- 鲁棒约束：对每个约束函数 \(g_i\)，通过管状边界 \(\tau_{i,k}\) 收紧名义约束：
  \[
  g_i(\zeta_k, v_k) + b_i + \tau_{i,k} \leq 0
  \]
  其中 \(\tau_{i,k}\) 是线性化约束梯度与响应矩阵乘积的范数和。
- 此外，要求状态可达管 \(R^s_k \subseteq I\)（域内约束）。
- 优化目标：名义轨迹成本、终端成本、管大小正则化项之和。

**E. 定理1**：在共形预测的覆盖保证下，若每次闭环测试得分与校准分数可交换，且SLS控制器对所有扰动 \(w_k \in \mathcal{B}^{n_s}\) 鲁棒正确，则真实状态和控制以概率至少 \(1-\delta\) 保持在可达管内。

5. **开放科学**：提供网站、代码、视频，支持复现。

## 8. 不足与局限

1. **对Push-T任务的局限性**：依赖欧氏距离到潜态目标，可能无法捕捉流形上的测地距离，导致长时程接触任务性能差。作者承认需集成学习的时间距离度量。
2. **共形预测交换性假设**：要求闭环测试得分与校准得分可交换，这在策略分布漂移严重时可能不成立。尽管作者通过收集有代表性随机滚动来支持，但未提供交换性检验。
3. **缺乏强对抗或极端泛化场景测试**：未在模型失配严重、观测噪声大、或奖励函数变化的情况下评估鲁棒性。
4. **控制器增益预设**：SLS假设反馈增益为线性，对强非线性系统可能过于保守；未来可探索非线性SLS。
5. **计算资源限制**：仅使用单卡RTX 4090，未展示多卡或更大规模部署的可行性。
6. **训练数据依赖性**：离线数据集需要专家或覆盖良好的策略，在完全未知环境中可能难以获取。
7. **部分结果仅展示名义规划**：Push-T任务未进行鲁棒安全实验；同样，Rope硬件仅做了名义规划，未展示鲁棒安全过滤。
