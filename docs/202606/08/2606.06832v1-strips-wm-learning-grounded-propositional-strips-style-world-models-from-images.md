## 摘要
执行长程视觉操作的机器人观察到高维图像，但成功的计划依赖于与动作相关的事实：现在能做什么以及之后会发生什么变化。一个有用的规划表示应丢弃不相关的视觉细节，同时保留动作的适用性和效果。经典任务规划器通过具有前置条件和效果的符号操作符来利用这种结构，但从原始视觉体验中获得此类表示仍然具有挑战性。我们研究了一种视觉任务规划设置，其中机器人仅接收图像转换：当前图像、执行的高级动作以及结果图像。在测试时，给定起始图像和目标图像，机器人必须生成一系列达到目标的高级动作。为了解决这个问题，我们提出了STRIPS-WM，一个直接从视觉转换中学习基于图像的STRIPS风格世界模型的框架。STRIPS-WM首先从图像中归纳出一个有限的抽象转换图，然后学习隐式的二元谓词和每个动作标签的一个基础命题操作符。学习到的操作符形成了一个具有稀疏前置条件和添加/删除效果的符号动作模型。最后，将学到的谓词蒸馏到一个视觉编码器中，使经典规划能够直接从新的起始图像和目标图像进行。在视觉重排任务上的实验表明，STRIPS-WM在图像到规划的成功率上优于测试的视觉展开、潜在图搜索和潜在符号基线方法。

## Abstract
Robots performing long-horizon visual manipulation observe high-dimensional images, but successful plans depend on action-relevant facts: what can be done now and what changes afterward. A useful planning representation should discard irrelevant visual details while preserving action applicability and effects. Classical task planners exploit this structure through symbolic operators with preconditions and effects, but obtaining such representations from raw visual experience remains challenging. We study a visual task-planning setting in which a robot receives only image transitions: the current image, executed high-level action, and the resulting image. At test time, given a start image and a goal image, the robot must produce a sequence of high-level actions that reaches the goal. To address this problem, we introduce STRIPS-WM, a framework for learning image-grounded STRIPS-style world models directly from visual transitions. STRIPS-WM first induces a finite abstract transition graph from images, then learns latent binary predicates and one grounded propositional operator per action label. The learned operators form a symbolic action model with sparse preconditions and add/delete effects. Finally, the learned predicates are distilled into a visual encoder, enabling classical planning directly from novel start and goal images. Experiments on visual rearrangement tasks show that STRIPS-WM improves image-to-plan success over the tested visual rollout, latent graph-search and latent-symbolic baselines.

## 1. 核心问题与研究动机

- **问题**：机器人执行长程视觉操作时，观测到高维图像，但有效规划仅依赖于少量与动作相关的事实（如当前物体是否可抓取、动作后状态如何变化）。直接在像素层面规划需要预测未来长序列图像，效率低且包含大量无关细节。经典任务规划器（如 STRIPS）使用符号状态（谓词）和带前置条件/效果的操作符，能简洁描述动作模型，但如何从原始视觉经验中自动学习这种符号化表示一直是挑战。
- **动机**：开发一个能从“当前图像-执行动作-结果图像”三元组中自动学习图像锚定的 STRIPS 风格世界模型的框架，从而在测试时仅凭起始/目标图像即可完成经典规划。

## 2. 方法论

### 核心思想
提出 STRIPS-WM，分为三个阶段：
1. **图像锚定的任务图学习**：从视觉转换数据中学习离散的状态分配，构建抽象状态转换图。
2. **从任务图学习 STRIPS 操作符与谓词**：使用 CP-SAT 求解器联合学习每个抽象状态的二元谓词向量和每个动作标识符的接地命题操作符（含前置条件、添加/删除效果）。
3. **视觉谓词分类器与测试时规划**：将学到的谓词向量蒸馏到视觉编码器中，对新的起始/目标图像预测谓词状态，然后利用学到的操作符进行经典规划。

### 关键技术细节
- **任务图构建**：
  - 使用 JEPA 风格架构：学生编码器将当前图像映射为离散 FSQ 码，教师编码器（EMA 更新）提供目标后继码；动作条件潜动力学模型预测后继码；逆模型预测动作标识符。
  - 通过训练使视觉上相似但动作相关不同的状态被分配到不同离散码，从而得到抽象状态图 `G_hat = (S_hat, E_hat)`。
- **操作符学习（CP-SAT）**：
  - 为每个图节点 `s` 分配二元谓词向量 `x_s ∈ {0,1}^m`，为每个动作标识符 `a` 学习四个掩码：正/负前置条件、添加/删除效果。
  - 正证据约束：观测到的边 `(s,a,s')` 必须满足前置条件且效果符合 STRIPS 转换（添加/删除/惯性）。
  - 负证据约束：对可信缺失的状态-动作对（即节点上未观测到的动作），要求至少违反一个前置条件，否则使用松弛变量。
  - 使用词典序目标最小化：① 转换松弛（观测边必须解释）；② 适用性松弛；③ 效果稀疏性；④ 前置条件稀疏性。
- **视觉谓词分类器**：
  - 用 ConvNeXt-Tiny 骨干网络，以多标签二元交叉熵训练，将图像映射到谓词概率，在测试时二值化得到初始/目标谓词状态。
  - 输出维度 m（由 CP-SAT 阶段选择）。

### 算法流程
1. 收集视觉转换数据集 `D_O`。
2. 训练离散编码器与潜动力学模型，获得任务图 `G_hat`。
3. 使用 CP-SAT 求解器为每个节点分配谓词向量，为每个动作学习操作符 `ω_a`。
4. 训练视觉谓词分类器 `b_ϕ`。
5. 测试时：输入 `o_start`、`o_goal` → `b_ϕ` 得到 `x_start`、`x_goal` → 在谓词状态下用学到的操作符进行经典规划（如 BFS），返回动作序列。

## 6. 主要结论与发现

- STRIPS-WM 在所有三个领域的图像到规划成功率上显著优于基线，尤其长程任务。
- BlocksWorld 上达到 100% 成功率，对应零松弛（精确学习），验证了理论保证（定理 10）。
- DinnerTable Real 上同样 100% 成功，尽管有松弛（将过度分裂的视觉状态合并为正确谓词状态）。
- DinnerTable 模拟域中因适用性松弛导致长程任务偶尔失败，但仍然远优于基线。
- 负证据（可信缺失动作）是学习有用前置条件的关键；没有负证据时，学习到的模型允许大量虚假适用的动作，规划失败。
- 谓词表示比直接图搜索更优，因为它能合并等价状态、修复不一致性，且感知预测维度更低。
- 总体证明：从图像直接学习可泛化的符号操作符是可行的，为视觉任务规划提供了有效途径。

## 7. 主要优点

- **端到端学习 STRIPS 世界模型**：无需物体标注、分割、手工定义谓词，仅需图像-动作-图像三元组。
- **三阶段分解清晰**：图构建、组合优化学习操作符、视觉分类器蒸馏，每一步可单独解释和调试。
- **CP-SAT 形式化**：利用组合搜索同时学习谓词分配和操作符，提供完备性保证（零松弛时等价于所有约束满足）。
- **稀疏性偏好**：词典序目标鼓励学习稀疏效果和前置条件，符合操作实际。
- **对视觉噪声鲁棒**：在真实图像上（DinnerTable Real）仍能恢复正确的抽象状态数，并成功规划。
- **消融实验充分论证负证据必要性**：设计干净，说服力强。
- **与经典规划器兼容**：学到的操作符可直接用于现有规划器（如 BFS、A* 等）。

## 8. 不足与局限

- **确定性假设**：模型假设确定性 STRIPS 动力学，无法处理随机结果或隐状态。
- **需要良好图覆盖**：视觉图的质量和覆盖度直接影响操作符精度；不完整动作标签或视觉模糊可能导致错误前置条件。
- **命题而非提升形式**：操作符是接地命题（grounded propositional），不是一阶模式（lifted first-order schemas），因此不能泛化到新物体或新参数（需更多数据重新学习）。
- **规模限制**：CP-SAT 求解在谓词数 m 和状态数 |S_hat| 较大时可能成为瓶颈（300 秒预算可能导致次优解）。
- **实验规模有限**：仅三个领域，一个真实场景，物体数量少；未评估大规模、高复杂度域。
- **计算资源未明确**：可复现性不足；没有报告训练时间或 GPU 规格。
- **感知对齐依赖**：视觉分类器的准确性直接影响测试时规划；虽然指标高，但对分布外图像（光照、视角变化）的鲁棒性未测试。
- **未与最先进潜世界模型（如 DINO-WM、Dreamer）直接比较**：文中提及但它们要么预测图像要么使用连续潜空间，但缺乏横跨符号抽象的系统对比。
