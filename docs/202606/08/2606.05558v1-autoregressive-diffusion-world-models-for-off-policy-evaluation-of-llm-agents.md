## 摘要
在多轮交互环境中评估大语言模型（LLM）智能体既昂贵又具有风险，因为这需要在线环境交互。我们提出ADWM（自回归扩散世界模型），一种仅从预收集轨迹中估计新LLM智能体策略性能的评估框架。其核心思想是学习一个潜在扩散世界模型，模拟环境如何响应评估策略，而无需在真实环境中执行该策略。现有的基于扩散的OPE方法通过联合扩散状态和动作来一次性指导完整轨迹，这一假设对于LLM智能体而言不成立，因为其动作是从观察环境后的策略中采样的离散文本。与因累积误差而受损的自回归世界模型不同，ADWM将每个转移建模为独立的去噪过程，实现了可靠的逐步展开，其中世界模型和智能体以因果顺序交替进行。关键之处在于，待评估的LLM智能体通过策略条件评分函数直接引导每一步的扩散生成，确保模拟轨迹准确反映其决策模式。实验表明，ADWM在多种多轮智能体任务中实现了准确的价值估计和评估可靠性，展示了其作为离线LLM智能体评估实用框架的潜力。

## Abstract
Evaluating large language model (LLM) agents in multi-turn interactive environments is expensive and risky, as it requires online environment interaction. We propose ADWM (Autoregressive Diffusion World Model), an evaluation framework that estimates the performance of a new LLM agent policy purely from pre-collected trajectories. The core idea is to learn a latent diffusion world model that simulates how the environment responds to the evaluation policy, without ever executing it in the real environment. Existing diffusion-based OPE methods guide full trajectories in a single pass by jointly diffusing states and actions, an assumption that breaks down for LLM agents whose actions are discrete text that must be sampled from the policy after observing the environment. Unlike autoregressive world models that suffer from compounding errors, ADWM models each transition as an independent denoising process, enabling reliable step-by-step rollouts where the world model and agent alternate in causal order. Crucially, the LLM agent under evaluation directly guides the diffusion generation at each step via a policy-conditioned score function, ensuring that simulated trajectories accurately reflect its decision-making patterns. Empirically, ADWM achieves accurate value estimates and evaluation reliability across diverse multi-turn agent tasks, demonstrating its promise as a practical framework for offline LLM agent evaluation.

### 1. 论文的核心问题与整体含义（研究动机和背景）

**研究动机**：大型语言模型（LLM）智能体在多轮交互环境（如网页导航、代码执行、长文本推理）中的部署日益增多，但其在线评估成本高昂（消耗API预算）且可能造成不可逆风险。传统评估需要直接在真实环境中执行待评估策略。

**核心问题**：如何仅从预先收集的离线轨迹数据中，准确估计一个新LLM智能体策略的价值，而无需与真实环境交互？这属于离线策略评估（Off-Policy Evaluation, OPE）问题。

**现有瓶颈**：
- 重要性采样方法：权重随轨迹长度指数增长，在LLM长序列任务中失效。
- 基于价值的方法（如FQE）：当评估策略与行为策略差异大时，偏差累积。

- 自回归世界模型：逐词生成导致误差逐步骤和逐词累积（复合误差）。

**整体含义**：论文提出一种新的“自回归扩散世界模型”（ADWM），将每个转移建模为独立去噪过程，避免复合误差，并通过策略条件评分函数让评估策略引导每一步生成，从而可靠地模拟轨迹进行离线评估。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

**核心思想**：将全局策略引导的轨迹律精确分解为一系列单步条件概率的乘积（定理1），每个单步条件概率由三个因子构成：先验、动作后验、策略延续因子。将该分解映射到引导扩散过程中，让评估策略通过其对数似然梯度驱动每一步的去噪，无需重训世界模型。

**关键技术细节**：
- **潜在空间与策略接口**：使用编码器`E`将原始文本观测映射为潜在表示`z_t`，再用投影器`G_ψ`生成软令牌`˜o_t`，供LLM策略在自身嵌入空间读取。编码器与所有组件端到端联合训练，确保表征反映动态而非表面语义；投影器通过InfoNCE对比损失+均方误差正则化进行训练。
- **引导分数函数（公式9）**：由三项之和构成：
  - 先验分数：来自得分网络的无条件输出。
  - 动作后验分数：通过无分类器引导（CFG）从条件与无条件得分之差恢复，对应`ω`参数。
  - 延续分数：将当前去噪估计`ˆz_{t+1}`展开为未来轨迹，反向传播评估策略`π_e`的对数似然梯度，并用`η_k`自适应加权（在去噪后期加强）。
- **价值估计**：通过蒙特卡洛展开生成轨迹，每步由LLM策略采样动作，世界模型生成下一个潜在表示，奖励头预测奖励，结局头判断终止，后对所有展开轨迹的折扣回报求平均（公式10）。

**算法流程**：
1. 训练阶段：在离线数据集上联合训练编码器、得分网络、奖励头、结局头、逆动态头等，同时训练投影器`G_ψ`。
2. 推理阶段：对每个待评估的LLM策略`π_e`，初始从行为池随机采样一个观测，交替执行：软令牌投影 → 策略采样动作 → 扩散采样下一潜在（使用引导分数） → 预测奖励和终止概率，直至终点或最大步数。多次展开后平均回报即为估值。
