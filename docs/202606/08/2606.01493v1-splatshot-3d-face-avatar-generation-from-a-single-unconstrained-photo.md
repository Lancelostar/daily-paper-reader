## 摘要
从单张非约束照片重建逼真的3D人脸头像具有挑战性：前馈式3D高斯泼溅（3DGS）模型在分布外输入上性能下降，而预训练扩散模型虽能生成高保真图像但缺乏多视角一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片真实感。基于此，我们提出SplatShot，一种无需训练的框架，直接在去噪过程中耦合这些表示。给定一个基础3DGS人脸模型和一张参考图像，我们通过每步3D反馈循环联合去噪所有目标视角。在每个时间步，从噪声潜变量预测干净图像，将这些多视角预测重新拟合3DGS，并将3DGS重渲染与2D预测之间的光度差异反向传播到噪声估计中。这引导采样轨迹朝向严格3D一致、身份保真的输出。在多样化的真实世界图像上的实验表明，SplatShot生成的3D头像具有卓越的身份保持、照片真实感和多视角一致性。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

- **现有方法的矛盾**：
  - **前馈式3D高斯泼溅（3DGS）**：直接从单张图像预测3DGS，速度快但严重依赖训练数据分布，对“分布外”的野外照片泛化差。
  - **预训练扩散模型**：生成单张图像质量高，但各视图独立采样，缺乏多视图一致性。
- **关键洞察**：显式3D表示保证几何一致性，2D扩散先验保证照片真实感，两者本质互补，应联合使用。SplatShot在**无需额外训练**的前提下，在扩散去噪过程中引入3D反馈循环，使两者相互校正。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
在扩散模型的每步去噪中，从当前噪声潜变量预测多视图干净图像，用这些图像更新（重拟合）一个基础3DGS模型，再将该3DGS的重渲染结果与扩散预测之间的光度差异**反向传播**到噪声估计中，从而引导采样轨迹朝向3D一致、身份保真的方向。

### 关键技术细节
- **整体流程**（图3）：
  1. 自动选择一个最匹配输入照片的“基础”3DGS模型（从NeRSemble预训练库中选出）。
  2. 从基础模型渲染多视图，编码到扩散潜空间，加入噪声至起始时间步 t_s（部分噪声而非纯噪声，保留结构）。
  3. 从 t_s 到 0 迭代执行：
     - **Step 1: 每视图扩散生成**：利用IP-Adapter以输入照片为条件，独立预测每视图的噪声，恢复预测的干净图像。
     - **Step 2: 3DGS更新**：用所有预测图像对3DGS进行多步微调（光度损失L1+SSIM），使3DGS逐渐容纳身份特征并解决视图间不一致。
     - **Step 3: 噪声调整**：计算重渲染与预测图像的光度损失 g，通过链式法则求梯度 ∂g/∂ϵ 并减去，调整噪声估计，使下一时间步的采样偏向3D一致性。
  4. 最终输出显式3DGS模型，可渲染任意新视角。

- **噪声混合机制（关键稳定策略）**：
  - 早期去噪步骤（大 t）中，模型预测的干净图像严重失真。若直接用于指导3DGS，会导致参数崩溃。
  - 将**模型预测噪声**与**添加时使用的真实噪声**按权重 w 混合：`ϵ_mix = w·ϵ_θ + (1-w)·ϵ_gt`。
  - 混合噪声代入公式预测干净图像 `x̂_0`，既保持梯度可导（来自预测部分），又获得稳定结构（来自真实部分），避免3DGS早期退化。

- **具体参数**：
  - 基础扩散模型：Stable Diffusion v1.5（Realistic Vision V4.0）
  - 身份条件：IP-Adapter Plus Face
  - 姿态控制：ControlNet（OpenPose）
  - DDIM步数：50，去噪强度 s=0.6（起始 t_s=600）
  - 噪声混合权重 w=0.4，3DGS指导权重 λ=50000
  - 3DGS重拟合：首次1000次迭代，后续每步200次
  - 视图数 V=16，背景黑色

## 7. 优点：方法或实验设计上的亮点

1. **训练-free**：无需微调任何网络组件，直接利用预训练扩散模型和现成3DGS基础模型，易于复现和推广。
2. **创新性耦合**：将显式3D表示（3DGS）与隐式生成先验（扩散）在去噪过程中实时相互指导，而非SDS的简单分数蒸馏。
3. **噪声混合机制**：巧妙解决早期去噪预测退化问题，使3DGS拟合稳定，是方法落地的关键设计。
4. **显式输出**：最终产物是完整的3DGS模型，可实时渲染任意新视角，而非扩散多视图集合。
5. **全面且公平的评估**：覆盖多个基线类别、多种指标、消融实验、人口公平性、失败案例，并指出对比方法的潜在不一致性（如GAGAvatar的3D不一致），实验设计严谨。

## 8. 不足与局限

1. **速度慢**：每个身份约3分钟，远慢于FastAvatar（3秒）等前馈方法，限制了实时/批量应用。
2. **姿势覆盖有限**：基础模型来自NeRSemble（前180°），未测试背面视图生成。
3. **发型继承问题**：身份条件（IP-Adapter）主要关注脸部，发型和服装大量继承自基础模型，若输入与基础发型不匹配则产生差异。
4. **特殊配件困难**：帽子等大型装饰物难以处理（原因：基础模型缺乏此类数据，且身份编码器可能将帽子视为身份的一部分而非可移除附件）。
5. **可扩展性**：基础模型库需要覆盖足够多样的几何结构（如发型、脸型），否则选择算法可能找不到合适初始。
6. **无更快的替代方案**：尝试PeRFlow流匹配加速（4步采样）失败，因为少量指导步骤不足以解决多视图不一致。
7. **缺乏背面与毛发细节**：3DGS表示在头发等半透明精细结构上仍有限。
