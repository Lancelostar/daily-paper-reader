## 摘要
预测现实世界事件需要语言模型代理在不完整、有时间限制的信息下进行不确定性推理。然而，评估代理是否真正进行预测不仅仅需要最终答案的准确性：模型可能通过回忆记忆中的训练事实、引用捏造的证据或产生无因果支持的故事而正确。我们提出了WorldReasoner，一个用于时间有效事件预测的评估框架。每个任务向代理提供一个已解决预测问题、一个模拟预测日期，并且只允许访问该日期之前可用的证据；解决后，该框架对提交的概率、引用的证据以及可选的因果事件图进行评分。WorldReasoner报告三个互补维度：针对已解决答案的结果质量、针对引用来源的证据质量以及针对解决后的后见之明图的推理质量。该基准由代理构建流程生成，该流程生成预测问题，收集带时间戳的证据，并大规模构建后见之明参考图，最终从14,141篇文章中产生345个已解决任务，这些文章中的图覆盖了8,087个提取的事件。在六种受控代理设置中，时间有效检索是结果准确性的最强驱动因素；因果图构建提高了关键事件的恢复；基于图的正确预测更牢固地以关键事件和相关来源为基础，但代理仍然难以将有根据的证据转化为校准的概率。

## Abstract
Forecasting real-world events requires language-model agents to reason under uncertainty from incomplete, time-bounded information. Yet evaluating whether agents genuinely forecast requires more than final-answer accuracy: a model may be correct by recalling memorized training facts, citing fabricated evidence, or producing an unsupported causal story. We present WorldReasoner, an evaluation framework for temporally valid event forecasting. Each task gives an agent a resolved forecasting question, a simulated forecast date, and access only to evidence available before that date; after resolution, the framework scores the submitted probability, cited evidence, and optional causal event graph. WorldReasoner reports three complementary axes: outcome quality against resolved answers, evidence quality over cited sources, and reasoning quality against post-resolution hindsight graphs. The benchmark is built by an agentic construction pipeline that generates forecasting questions, collects time-stamped evidence, and builds hindsight reference graphs at scale, yielding 345 resolved tasks derived from 14,141 articles with graphs covering 8,087 extracted events. Across six controlled agent settings, temporally valid retrieval is the strongest driver of outcome accuracy; causal graph construction improves key-event recovery; and correct graph-enabled forecasts are more strongly grounded in key events and relevant sources, yet agents still struggle to convert grounded evidence into calibrated probabilities.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有LLM预测评估仅关注最终答案准确性，无法区分模型是真正基于有效推理进行预测，还是通过记忆训练数据中的事后事实、引用捏造证据或构建无因果支持的故事来“蒙对”。
- **背景**：
  - 静态基准（如Autocast）存在时间泄漏——模型可能已从训练语料中见过事后信息，答案准确率不可靠。
  - 实时基准（如RealTime QA、FutureX）能保证时间边界，但评估慢、难扩展、不可复现。
  - 多数基准只评分最终答案，忽视证据和推理过程。
- **意义**：需要一种既能避免时间泄漏、又可复现，且能从结果、证据、推理三个维度客观评估预测代理的框架，以判断其是否真正进行有效推理。

## 2. 论文提出的方法论
- **核心思想**：构建一个基于模拟历史日期的可复现预测评估框架，通过“时间门控”和“事后因果图”实现三维度评估。
- **关键技术细节**：
  - **任务定义**：每个任务包含问题文本、最早可预测日期、模拟预测日期、解决日期和已解决结果。代理在模拟日期只可访问该日期之前的证据（时间门控）。
  - **代理提交**：预测答案、概率、引用来源，以及可选的因果预测图（带节点和边）。
  - **后见之明参考图构建**（后向管道）：在问题解决后，由HindsightAgent收集证据并合成因果解释，再由GraphBuilderAgent转换为结构化图，包含事件节点（带时间戳）、有向因果边（带类型、强度、置信度）和结果影响记录（方向、大小、置信度）。自动质量检查（如图深度≥3、节点数≥10、来源数≥20等）并允许自我修正。
  - **三维度评估**：
    - **结果质量（Outcome quality）**：准确率、Brier分数、对数分数。
    - **证据质量（Evidence quality）**：来源精确度（SrcP）= 引用来源中出现在后见之明证据集中的比例。
    - **推理质量（Reasoning quality）**：关键事件召回率（KER）= 预测图事件中成功匹配后见之明图的前K个（K=5）关键事件的比例，匹配需同时满足文本相似度（BM25+词级F1混合）和日期临近（7天内）。
  - **污染过滤**：排除问题开始时间早于模型知识截止日期的样本，避免模型从训练数据中记忆答案。
- **算法流程**（文字说明）：
  1. 前向管道从预测市场和新间流生成问题，经质量筛选存入数据库。
  2. 后向管道在问题解决后，调用LLM构建后见之明参考图（证据收集→图构建→质量检查→修正）。
  3. 评估时，将模拟日期给代理，通过时间门控暴露该日期之前的证据。
  4. 代理提交预测，系统自动计算三个轴分数。

## 6. 主要结论与发现
1. **时间有效检索是结果准确性的最大驱动因素**：Search-Enabled的加权平均准确率68.8%，显著高于Vanilla的58.7%和Causal Simulation的56.6%。
2. **因果图构建提升推理质量但不提升准确性**：SE Graph的KER（9.8%）高于Causal Simulation（4.9%），但准确率（64.4%）反而低于Search-Enabled（68.8%）。说明图构建帮助识别关键事件，但最终决策可能因注意力分散或概率校准失败而未能受益。
3. **正确预测与更好的证据/推理相关性**：在SE Graph条件下，正确预测的SrcP（62.6% vs 48.0%）和KER（10.8% vs 8.0%）均高于错误预测，但分离不完全，表明仍存在“检索到相关证据但无法转化为校准概率”的瓶颈。
4. **污染过滤显著改变模型排名**：如GPT-5.4未过滤准确率69.2%，过滤后仅52.7%，说明缺乏时间约束时容易高估能力。
5. **时间敏感性正确**：随模拟日期接近解决，准确率上升（从Early约62%到Near-Res约77-80%），验证了基准的难度梯度合理。

## 7. 优点（方法或实验设计亮点）
- **三维度评估**：首次将结果、证据、推理分离，能区分“正确但基于错误证据”和“错误但基于相关证据”等不同失败模式，比单一准确率更有诊断价值。
- **时间有效性控制**：通过模拟历史日期+时间门控+污染过滤，同时保证了可复现性和时间边界，解决了静态基准的时间泄漏和实时基准的不可复现问题。
- **自动构建管道**：前向和后向管道可扩展，从新闻和预测市场自动生成任务和后见之图，支持持续扩大基准规模。
- **人工验证**：对后见之图事件进行人工注释，确保参考图质量（68.3%通过率，主要失败模式为来源不匹配12.2%）。
- **实验设计全面**：覆盖从无检索到实时检索的渐进设置，并考虑了不同模型、时间点、语义匹配鲁棒性等。

## 8. 不足与局限
- **覆盖偏差**：基于新闻和预测市场，偏向覆盖度高的公众话题（政治、文化、健康占58%），其他领域（如科学、商业）较少；快节奏公共话题过采样。
- **分辨率与时间粒度**：部分问题在主观或制度相关领域存在模糊解决；天级别时间门控可能遗漏日内信息泄漏。
- **模型知识截止日期不完美**：截止日期是训练数据存在的粗略代理，无法精确反映模型参数中是否包含事后信息。
- **后见之图质量近似**：仅人工注释了固定子集（492事件），因果边强度和影响分数由LLM赋予，可能继承模型偏差；精确来源匹配可能低估实际相关证据。
