## 摘要
我们提出了TriFlow，一种新的生成方法，可直接从输入几何条件（如有符号距离场）生成具有类似艺术家三角形拓扑的紧凑3D网格。我们的关键洞察是将网格拓扑表示为在表面上定义的最近顶点向量场（NVF），其中每个点编码其在局部重心坐标中与最近三角形顶点的关联。我们训练一个潜在流匹配模型来合成该场，从而实现基于输入几何条件的拓扑生成。为了提取连贯的网格，我们使用生成的NVF对表面区域进行聚类，并通过拓扑感知优化引导约束二次误差度量（QEM）网格简化。这产生了与输入几何紧密匹配且具有结构化、类似艺术家连接性的输出网格。实验表明，与最先进的基于学习的方法相比，TriFlow实现了更强的泛化能力和显著改善的拓扑质量，同时查默斯距离降低了90%，速度提升了8倍。

## Abstract
We present TriFlow, a new generative approach for producing compact 3D meshes with artist-like triangle topology directly from input geometry conditions such as signed distance fields. Our key insight is to represent mesh topology as a nearest-vertex vector field (NVF) defined over the surface, where each point encodes its association to the nearest triangle vertex in the local barycentric frame. We train a latent flow-matching model to synthesize this field, enabling topology generation conditioned on the input geometry. To extract a coherent mesh, we cluster surface regions using the generated NVF and guide a constrained quadric error metric (QEM) mesh simplification with topology-aware optimization. This yields output meshes that closely match the input geometry while exhibiting structured, artist-like connectivity. Experiments demonstrate that TriFlow achieves stronger generalization and significantly improved topology quality compared to state-of-the-art learning-based approaches, alongside 90% lower Chamfer Distance and an 8x speedup.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：从隐式场（如SDF）或生成模型输出中提取的3D网格通常存在过度细碎、拓扑杂乱、非流形的缺点，无法直接用于游戏、影视等生产管线。现有基于自回归序列生成的方法（如MeshAnything、TreeMeshGPT）虽然能生成一定规律的网格，但存在逐token推理慢、误差累积、泛化性差等问题。
- **目标**：提出一种**非自回归**的生成方法，直接从输入几何条件（如SDF）生成紧凑、流形、类似人类艺术家手动建模（artist-like）的三角形网格拓扑，同时保持高几何保真度。
- **核心洞察**：网格拓扑可以表示为一个定义在表面上的**分段连续向量场**，而非离散的连接关系。利用这一表示，将拓扑生成转化为**向量场建模**问题，从而避开自回归序列的缺陷。

## 2. 方法论：核心思想、关键技术细节

### 2.1 最近顶点向量场（NVF）
- **定义**：给定表面上的任意点 \(p\)，NVF \( \mathbf{t}(p) \) 指向包含该点的三角形中重心坐标最大的顶点（即最近顶点）：
  \[
  \mathbf{t}(p) = v_n - p, \quad n = \arg\max_{i\in\{1,2,3\}} \lambda_i
  \]
  其中 \((\lambda_1,\lambda_2,\lambda_3)\) 是点 \(p\) 在局部三角形中的重心坐标。
- **性质**：该场将每个三角形自然划分为三个区域，对应三个顶点；区域之间的邻接关系隐含了网格的顶点连接性。场是分段连续的，适合神经网络建模。
- **体素化**：为便于训练，将NVF离散化到512³稀疏体素网格上，每个体素存储指向目标顶点的向量。

### 2.2 潜在流匹配模型
- **输入条件**：输入几何以SDF表示，通过一个SDF自编码器（fine-tune自Direct3D-S2的VAE）编码为64³的稀疏潜在 \(z_{\text{SDF}}\)。
- **NVF编解码**：NVF同样使用VAE（相同架构）编码为64³潜在 \(z_{\text{NVF}}\)。输出时参数化为方向 \(\hat{\mathbf{d}}\) 和幅度的平方根 \(\hat{s}\)，以增强靠近顶点处方向的连续性。
- **条件生成**：训练一个基于TRELLIS架构的**潜在流匹配模型**，以 \(z_{\text{SDF}}\) 和用户控制参数（目标面数、四边形比例）为条件，从噪声逐步回归到真实的 \(z_{\text{NVF}}\)。
- **训练目标**：条件流匹配损失（L1范数），在噪声与真实潜在之间线性插值路径上学习速度场。

### 2.3 从NVF提取网格
1. **NVF平滑与传递**：对预测的NVF进行双边滤波降噪；然后通过Marching Cubes从输入SDF提取一个过密网格 \(M_d\)，将体素化NVF传递到其顶点上。
2. **Watershed算法**：
   - 以预测位移幅度小的顶点（即靠近目标顶点的点）作为**根顶点**，每个根是一个初始簇。
   - 基于优先级队列，根据预测目标位置的欧氏距离向邻居传播簇标签，进行类似分水岭的迭代扩展。
   - 更新NVF，使所有顶点指向其所属根顶点的预测位置。
3. **约束QEM简化**：
   - 在QEM边缘折叠中，**拒绝**属于不同簇的顶点之间的折叠，保持拓扑结构。
   - 当折叠涉及根顶点时，在几何二次型中加入**位置约束二次型** \(Q_t\)，使收缩点偏向生成模型预测的目标位置，权重视参数 \(\lambda_t\) 控制。
4. **几何数据增强**：训练时对表面施加随机变形场，使模型对噪声和异常几何具有鲁棒性。

## 7. 优点
- **表示创新**：NVF将离散拓扑编码为分段连续场，使生成问题适合流匹配等连续生成模型，规避了序列模型的固有问题。
- **非自回归，高效**：无需逐token生成，一次前向推理即可得到场，再通过并行优化快速提取网格。
- **鲁棒性强**：数据增强使模型能处理噪声几何（如生成模型输出）；Watershed+约束QEM能稳定提取拓扑。
- **可扩展性**：NVF分辨率可独立提高（如附录1024³），有望处理更细节特征。
- **用户控制**：支持目标面数和四边形比例条件，适应不同LOD需求。
- **充分的消融和感知验证**：不仅对比基线，还通过人类评估和VLM评估佐证了拓扑质量。

## 8. 不足与局限
- **体素依赖**：当前NVF训练和推理依赖512³体素网格，限制了处理大规模场景的能力（单个物体在小场景内可行，但大场景效率低）。未来可通过多分辨率或分块策略解决。

- **拓扑类型受限**：主要针对三角形网格，未显式处理四边形或混合网格，但通过四边形比例参数可一定程度上控制规则性。
- **对输入SDF质量敏感**：虽然数据增强缓解了部分噪声，但极端破损或不完整的SDF仍可能影响NVF预测。
