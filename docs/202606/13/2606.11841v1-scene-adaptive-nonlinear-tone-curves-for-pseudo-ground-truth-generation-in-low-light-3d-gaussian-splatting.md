## 摘要
低光照新视角合成具有挑战性，因为暗光多视图图像包含噪声、微弱的结构细节和压缩的动态范围。最近的3D高斯泼溅（3DGS）方法通过在无法获得配对的正常光照参考时生成伪真值（pseudo-GT）图像作为监督目标来应对这些挑战。现有的伪真值方法对所有像素应用统一的线性增益，这会裁剪明亮区域，同时对暗区域增强不足，限制了重建质量。我们观察到，在二维低光照增强中早已建立的非线性色调映射尚未被探索用于三维重建中的伪真值生成。因此，我们提出一个场景自适应的非线性色调曲线框架，用非线性替代方案取代线性伪真值。该框架引入了基于百分位数的归一化以实现场景无关的曲线应用、场景自适应偏移以自动调整黑电平，以及两条互补曲线：自适应SoftExp（ASE），一条有界指数曲线，和自适应Poly3（AP3），一条数据驱动的三次多项式。该模块仅改变伪真值的计算，而保持3DGS主干不变。在覆盖21个场景的三个基准上的实验表明，两条曲线均持续优于线性基线，在LOM上PSNR提升高达+4.34 dB，在RealX3D上提升+3.25 dB。尽管数学形式不同，两条曲线实现了相似的性能，表明改进与曲线无关。代码可在https://github.com/lvmingzhe/adaptiveToneCurve获取。

## Abstract
Low-light novel view synthesis is challenging because dark multi-view images contain noise, weak structural detail, and compressed dynamic range. Recent 3D Gaussian Splatting (3DGS) methods address these challenges by generating pseudo ground-truth (pseudo-GT) images as supervision targets when paired normal-light references are unavailable. Existing pseudo-GT methods apply a uniform linear gain to all pixels, which clips bright regions while providing insufficient enhancement in dark regions, limiting reconstruction quality. We observe that nonlinear tone mappings, long established in 2D low-light enhancement, have not been explored for pseudo-GT generation in 3D reconstruction. Accordingly, we propose a scene-adaptive nonlinear tone-curve framework that replaces linear pseudo-GT with nonlinear alternatives. The framework introduces percentile-based normalisation for scene-agnostic curve application, a scene-adaptive offset for automatic black-level adjustment, and two complementary curves: Adaptive SoftExp (ASE), a bounded exponential curve, and Adaptive Poly3 (AP3), a data-driven cubic polynomial. The module changes only the pseudo-GT computation and leaves the 3DGS backbone unchanged. Experiments on three benchmarks covering 21 scenes show that both curves consistently outperform the linear baseline with PSNR improvements up to +4.34 dB on LOM and +3.25 dB on RealX3D. Both curves achieve similar performance despite their different mathematical forms, suggesting the improvement is curve-agnostic. Code is available at https://github.com/lvmingzhe/adaptiveToneCurve

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：低光照新视角合成（NVS）中，暗光多视图图像包含噪声、弱结构细节和压缩的动态范围。现有方法（如LITA-GS）通过生成伪真值（pseudo-GT）图像来监督训练，但普遍使用**均匀线性增益**（将低光照帧乘以一个固定增益因子后裁剪到[0,1]）。线性策略存在两个固有局限：
  - 统一放大导致亮部过曝（clipping），丢失梯度信号；
  - 无法根据亮度区域自适应调整，暗区增强不足。
- **背景**：2D低光照增强中**非线性色调映射**已被长期验证有效，但尚未被引入3D重建的伪真值生成。本文发现该设计空白，并证明非线性曲线可显著提升重建质量，且改进具有**曲线无关性**（两种不同形状的曲线性能接近）。

## 2. 方法论

- **核心思想**：用**场景自适应非线性色调曲线**替换线性伪GT，作为3DGS训练中的监督目标。模块仅修改伪GT计算，不改变LITA-GS主干网络、损失函数或渲染器。
- **关键技术细节**：
  - **百分位归一化**：用输入图像的亮度通道Y的p90（90%分位）和p10（10%分位）取代均值归一化，增强对异常值的鲁棒性，使曲线形状场景无关。
  - **场景自适应偏移**（δ₀）：基于阴影致密比 \( r = p_{10} / p_{90} \) 计算 \( \delta_0 = \alpha (1 - r)^\beta \)，其中α=0.12, β=2.5（通过网格搜索在BlueHawaii场景上确定，跨数据集固定）。r小说明阴影压缩严重，需更大偏移提升黑电平。
  - **两条互补曲线**：
    - **Adaptive SoftExp (ASE)**：有界指数曲线 \( f_{ASE}(x) = \delta_0 + (1-\delta_0)(1 - e^{-g \cdot t}) \)，其中 \( t = x/p_{90} \)，增益g随输入亮度自适应：\( g = g_0 + c \cdot \bar{Y} \)（\(\bar{Y}\)为平均亮度）。自然有界，C∞光滑。
    - **Adaptive Poly3 (AP3)**：数据驱动三次多项式 \( p(t) = 0.072t^3 - 0.438t^2 + 0.995t \)，系数从BlueHawaii场景配对低/正常光图像回归得到并冻结。最终 \( f_{AP3}(x) = clamp(\delta_0 + p(t)^b, 0, 1) \)，其中 \( b = 1 + c \cdot \bar{Y} \) 为亮度自适应缩放因子。近线性初始斜率（≈1），三次滚降防止过饱和，需clamp（<1%像素受影响）。
  - **亮度自适应增益c**：唯一需数据集级调整的参数。对一致低光照（LOM、RealX3D）设c>0；对亮度变化范围大的数据（MipNeRF360-varying）设c=0。
- **集成方式**：将原线性伪GT \( \hat{I}_{in} = clamp(\theta / mean(I_{low}) \cdot I_{low}, 0, 1) \) 替换为 \( \hat{I}_{in} = f_{ASE}(I_{low}) \) 或 \( f_{AP3}(I_{low}) \)，计算开销<0.5ms/迭代。

## 6. 主要结论与发现

- **非线性映射是主要驱动因素**：两种形状不同的曲线（ASE和AP3）在所有基准上均显著优于线性基线，且彼此性能接近（LOM PSNR差0.17dB，RealX3D差0.68dB，MipNeRF360-varying差0.87dB），表明改进源于非线性本身而非特定参数化。
- **量化提升**：
  - LOM：AP3平均PSNR+4.34dB（24.43 vs 20.09），ASE+4.17dB；SSIM+0.04，LPIPS改善。
  - RealX3D：AP3 NVS PSNR+3.25dB（18.50 vs 15.25），ASE+2.57dB；AP3在8个留出场景上仍达18.02dB。
  - MipNeRF360-varying：ASE PSNR+1.79dB（17.95 vs 16.16），SSIM+0.026，LPIPS−0.035；AP3也有改善。
- **场景自适应偏移的必要性**：固定偏移（如+0.103）在bike场景（GT阴影几乎为0）导致过亮，PSNR下降；自适应偏移（δ₀=0.038）使曲线贴近GT，从退步转为提升。
- **分布重塑**：非线性曲线将低光照图像严重右偏的像素分布平滑重塑至接近GT的偏度（误差±0.3），使L1梯度更均匀。
