## 摘要
我们提出了MentalThink，一种视觉-符号推理范式，它为多模态大语言模型（MLLMs）配备了一种可执行的“思维”可视化机制。MentalThink的核心是一个基于SVG的思考流水线（think-with-SVG pipeline），其中模型学习生成、渲染和解释可缩放矢量图形（SVG）代码，作为多轮推理的中间视觉表示。通过创建结构化的矢量草图，模型可以外部化空间假设，通过确定性渲染检查这些假设，并在受限的几何空间内进行推理，有效模拟人类心理意象的过程。我们通过一个两阶段训练框架实例化这一范式，结合监督微调（SFT）以实现SVG句法对齐，以及多轮强化学习（RL）以鼓励对中间视觉假设进行迭代检查、修订和优化。大量评估表明，MentalThink在空间理解和推理基准测试（如VSIBench上55.1%，MindCube上76.0%）上取得了优越性能，表明可执行的矢量图形为动态视角转换、视觉反思和组合场景构建提供了一个可验证的视觉工作空间。

## Abstract
We introduce MentalThink, a visual-symbolic reasoning paradigm that equips Multimodal LLMs (MLLMs) with an executable mechanism for "mental" visualization. The core of MentalThink is a think-with-SVG pipeline, where the model learns to generate, render, and interpret scalable vector graphics (SVG) code as an intermediate visual representation for multi-turn reasoning. By creating structured vector sketches, the model can externalize spatial hypotheses, inspect them through deterministic rendering, and reason within a constrained geometric space, effectively mimicking the human process of mental imagery. We instantiate this paradigm through a two-stage training framework, combining Supervised Fine-Tuning (SFT) for SVG syntactic alignment with multi-turn Reinforcement Learning (RL) to encourage iterative inspection, revision, and refinement of intermediate visual hypotheses. Extensive evaluations demonstrate that MentalThink achieves superior performance on spatial understanding and reasoning benchmarks (e.g., 55.1% on VSIBench, 76.0% on MindCube), showing that executable vector graphics provide a verifiable visual workspace for dynamic perspective taking, visual reflection, and compositional scene construction.

- **核心问题**：多模态大语言模型（MLLMs）在空间理解和推理任务中缺乏类似人类的“心理意象”（mental imagery）能力，即无法在推理过程中动态构建、检查和修正内部视觉表示。现有方法要么依赖纯文本推理（如Chain-of-Thought），缺乏视觉基础，易产生语义合理但视觉错误的幻觉；要么使用显式布局表示（如2D/3D边界框、认知图）或外部工具，但会简化视觉细节或破坏端到端可微性。
- **整体含义**：本文旨在为MLLMs提供一种可执行的“心理可视化”机制，使其能够通过生成和操作结构化矢量图形（SVG）代码来模拟人类的心理意象过程，从而在复杂的多步空间推理中实现动态视角转换、视觉反思和组合场景构建，最终提升模型的空间智能。

## 2. 方法论：核心思想、关键技术细节与流程

- **核心思想**：提出“think-with-SVG”推理范式，将Scalable Vector Graphics（SVG）作为中间视觉表示，使模型能够在推理过程中生成、渲染和解释SVG代码，将其作为“心理草图”来外部化空间假设，并通过确定性渲染获得视觉反馈，形成一个“假设→渲染→反思→修正”的闭环。
- **技术细节**：
  - **SVG作为结构化视觉语言**：定义SVG代码空间 S = {σ: σ = ⟨E, A, H⟩}，其中E为基本元素（如<rect>, <circle>），A为属性（位置、尺寸、样式），H为层次结构。SVG具有确定性映射、组合语义和参数化控制三大优势。
  - **SVG增强推理状态**：扩展多模态CoT状态为 s_svg = (q, v, ξ_svg)，其中 ξ_svg = {(ℓ₁, σ₁), …, (ℓ_T, σ_T)}，ℓ为语言推理，σ为可选SVG代码。
  - **渲染环境**：SVG代码经解析→DOM树→布局→光栅化，生成像素级图像 I_σ = R(σ)，提供确定性视觉反馈。
- **训练流程**：

## 7. 优点

- **方法创新性**：首次将SVG作为中间推理模态，而非最终输出，赋予模型“心理草图”能力，结合了符号推理的精确性和视觉表示的丰富性。
- **两阶段训练框架合理**：先SFT对齐句法，再RL优化策略，循序渐进，避免了直接RL训练的不稳定性。
