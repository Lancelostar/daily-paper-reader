## 摘要
推理分割旨在根据复杂的语言查询预测目标的像素级掩码。现有方法利用多模态大语言模型进行视觉语言推理，并生成中间目标线索（如点或框）以引导分割模型。然而，将丰富的推理压缩为稀疏线索往往会引入模糊性和噪声，导致这些线索无法准确保留推理意图。虽然多种互补线索可以丰富目标信息，但现有方法通常将它们联合输入单一分割过程，使得模糊或错误的线索影响整个预测。因此，我们提出DGSeg，一个学习融合由语义和空间线索引导的预测的推理分割框架。具体而言，多模态大语言模型联合推理目标身份和空间位置，生成互补的语义和空间线索，并将其输入到不同的分割分支中。通过一个轻量级的动态门控模块（该模块通过相对分支质量监督进行训练）自适应地整合它们的预测，以抑制噪声或冲突区域。大量实验表明，DGSeg在多个基准上持续优于强基线，并在具有挑战性的ReasonSeg验证集和测试集上分别达到69.6%和67.3%的gIoU。代码可在https://github.com/RZZeng/DGSeg获取。

## Abstract
Reasoning segmentation aims to predict pixel-wise masks for targets given complex language queries. Existing approaches leverage Multimodal Large Language Models (MLLMs) for vision-language reasoning and generate intermediate target cues (e.g., points or boxes) to guide a segmentation model. However, compressing rich reasoning into sparse cues often introduces ambiguity and noise, preventing these cues from accurately preserving the reasoning intent. While multiple complementary cues can enrich target information, existing methods typically feed them jointly into a single segmentation process, allowing ambiguous or erroneous cues to affect the entire prediction. Therefore, we propose DGSeg, a reasoning segmentation framework that learns to fuse predictions guided by semantic and spatial cues. Specifically, the MLLM jointly reasons about both target identity and spatial location, producing complementary semantic and spatial cues that are fed into separate segmentation branches. Their predictions are adaptively integrated by a lightweight dynamic gating module trained with relative branch-quality supervision to suppress noisy or conflicting regions. Extensive experiments demonstrate that DGSeg consistently outperforms strong baselines on multiple benchmarks and achieves 69.6% and 67.3% gIoU on the challenging ReasonSeg validation and test splits. Code is available at https://github.com/RZZeng/DGSeg.

- **研究动机**：如何高效降低目标线索中的指向偏置（referential bias），使最终分割更好地对齐MLLM的推理意图。

## 2. 论文提出的方法论

### 2.1 核心思想
提出一种新的推理分割流水线：**独立处理互补线索 + 可学习的自适应融合**。将MLLM生成的语义线索（文本描述）和空间线索（边界框）分别输入两个分割分支，然后通过轻量级动态门控模块根据分支特征预测像素级融合权重，加权融合得到最终掩码。

### 2.2 关键技术细节

| 模块 | 描述 |
|------|------|
| **语义-空间线索生成** | MLLM（Qwen2.5-VL）输出推理链条，然后分别输出语义描述 `csem` 和边界框 `cspa`（语义+空间双线索） |
| **双分支分割** | 基于SAM3构建：语义分支用 `csem` 提示，空间分支用 `cspa` 提示，各自得到初始掩码 logits `ℓ_sem`、`ℓ_spa` |
| **动态门控模块** | 轻量网络（两层卷积），输入双分支的像素嵌入特征 `f_sem` 和 `f_spa`，输出融合权重图 `W`（大小与特征图一致，用 Sigmoid 归一化） |
| **融合预测** | 最终 logit `ℓ = W↑ ⊙ ℓ_sem + (1-W↑) ⊙ ℓ_spa`，再二值化得到最终掩码 |
| **监督信号** | 阶段2训练时，使用软融合目标 `W*`，其根据两分支掩码与真实掩码的 IoU 计算相对质量（带温度 softmax） |

### 2.3 两阶段训练策略

- **阶段1：RL 微调 MLLM**  
  - 冻结分割模型，用 GRPO 优化 MLLM 生成结构化输出（`<think></think>` 和 `<answer></answer>`）
  - 复合奖励：格式奖励 + 空间奖励（预测框与 GT 框的 IoU）+ 语义奖励（语义分支掩码与 GT 掩码的 IoU）
- **阶段2：融合训练**  
  - 冻结 MLLM 和 SAM3 骨干，仅训练动态门控模块
  - 损失函数：`L = L_seg + λ_w L_w + λ_ent L_ent`  
    - `L_seg`：标准分割损失（如 dice + bce）
    - `L_w`：预测权重与软目标 `W*` 的交叉熵
    - `L_ent`：熵正则化，避免权重过于平均
  - 对两分支不一致的像素施加更高权重，突出关键区域

## 8. 不足与局限

- **依赖 MLLM 线索质量**：尽管门控能抑制部分噪声，但若 MLLM 本身推理错误严重（如完全错误的语义或框），门控仍可能无法完全纠正。
- **分割骨干单一**：仅验证了 SAM3，未探索与其他提示式分割模型（如 SAM、SAM2）的兼容性。
- **训练数据规模较小**：仅使用 9,000 个 RefCOCOg 实例，可能限制泛化性和多样性。

- **未涉及视频或时序推理**：论文聚焦静态图像推理分割，对视频或多模态动态场景尚未评估。
