## 摘要
跨图像比较推理对于视觉语言模型（VLM）仍然具有挑战性，尤其是在正确预测需要细粒度属性定位和全局一致推理的情况下。我们提出了CoRe，一个针对该问题的统一框架。CoRe包括：（i）CoRe-20K，一个大规模基于三元组的训练集，通过多专家协作管道从结构化视觉元数据自动构建，涵盖计数、深度、距离和空间关系；（ii）TriSR，一个结构化奖励框架，在GRPO优化下联合监督属性定位、判断对齐和三元组一致性；（iii）CoRe-Bench，首个专门用于细粒度跨图像比较推理的基准。实验表明，CoRe在CoRe-Bench上显著优于现有VLM，同时在标准多模态基准上保持竞争力，在部分准确率上相比最强基线提高了28.2个百分点。

## Abstract
Cross-image comparative reasoning remains challenging for vision-language models (VLMs), especially when correct prediction requires fine-grained attribute grounding and globally consistent reasoning. We present CoRe, a unified framework for this problem. CoRe includes: (i) CoRe-20K, a large-scale triplet-based training set automatically constructed from structured visual metadata through a multi-expert collaborative pipeline, covering counting, depth, distance, and spatial relations; (ii) TriSR, a structured reward framework that jointly supervises attribute grounding, judgment alignment, and triplet consistency under GRPO optimization; and (iii) CoRe-Bench, the first benchmark dedicated to fine-grained cross-image comparative reasoning. Experiments show that CoRe substantially outperforms existing VLMs on CoRe-Bench while remaining competitive on standard multimodal benchmarks, achieving a 28.2-point gain in partial accuracy over the strongest baseline.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有视觉语言模型（VLM）在单图像理解任务上表现良好，但在需要**跨图像比较推理**的场景（如比较不同图像中的物体数量、深度、距离、空间关系）时表现严重不足。即使是最新的大型模型也频繁出错，且出错具有系统性。
- **背景**：现有基准（如 MME、CV-Bench）主要评测单图像感知；多图像基准（如 MuirBench）侧重语义整合，缺乏对细粒度属性比较的评测。此外，仅优化最终答案正确性会导致模型通过“捷径”或错误推理得到正确结果（CA+WR 比例最高达 17.1%），而使用 LLM 作为裁判评估推理轨迹存在与人类判断的对齐偏差。
- **本文目标**：系统性地研究并改进 VLM 的跨图像比较推理能力，提出包含专门数据集、训练框架和评测基准的统一方案。

## 2. 方法论
- **核心思想**：利用结构化视觉元数据自动构建高质量三元组训练数据，并设计结构化奖励框架，对推理过程中的可验证中间步骤（属性定位、判断一致性、三元组逻辑一致性）进行监督。
- **关键技术细节**：
  - **数据构建 – 多专家协作管道**：
    - **度量提取专家**：根据任务类型从元数据中提取统一度量（计数：实例数；深度：区域中位数深度；距离：欧氏距离；空间关系：边界框标签）。
    - **质量控制专家**：通过噪声下限（排除噪声）和难度上限（排除琐碎）过滤三元组，确保比较信号有意义；对分类任务加入多样性约束。
    - **问题生成专家**：从模板库生成三个成对比较问题，并随机化选项位置消除偏差。
  - **结构化奖励框架 TriSR**：
    - **属性定位奖励 R_ag**：鼓励模型正确估计每幅图像的属性值（使用指数负误差得分）。
    - **判断对齐奖励 R_ja**：同时检查推理链中的中间判断是否与 ground truth 一致以及与最终答案一致。
    - **三元组一致性奖励 R_tc**：检查三个成对比较结果是否满足全局逻辑一致性（例如区间可加性、序关系传递性），仅依赖模型自身预测，提供自一致性信号。
    - **总奖励**：组合任务准确性奖励（R_task）与三个结构化奖励，用权重 λ 控制。
  - **优化算法**：使用 **GRPO**（Group Relative Policy Optimization），对每个 query 采样多个响应，计算组内归一化优势，并加入 KL 正则化。
  - **输出解析**：强制模型输出结构化回答（<think>...</think><answer>...</answer>），通过规则解析提取属性估计、中间判断和最终答案。

## 3. 实验设计
- **数据集**：
  - **CoRe-20K**：自动构建的 20000 个比较三元组，覆盖计数、深度、距离、空间关系各 5000 个三元组（共 60000 个 QA 对）。其中 16000 个用于训练，4000 个用于评测（CoRe-Bench）。
  - **CoRe-Bench**：首个专用细粒度跨图像比较推理基准，包含 4000 个三元组（每任务 1000 个），回答格式为<answer>答案</answer>。
  - **CoRe-OOD-Binary**：使用外部数据集（UCF-QNRF、ARKitScenes、Argoverse 2、COCO）构建的二图像分布外基准，用于泛化性测试。
- **对比方法**：包括闭源模型（Claude3.7-Sonnet, GPT-4o）和多种开源 VLM（Qwen3-VL-4B/32B, InternVL3-2B/8B, LLaVA-OneVision-4B/8B, SSRL-4B, Remot-4B）。
- **评测指标**：
  - **Overall Accuracy**：仅当三元组中所有三个子问题全部正确时才计为正确。
  - **Partial Accuracy**：按子问题正确比例计分。
- **评估协议**：使用 VLMEvalKit 统一格式和答案提取规则。

## 6. 主要结论与发现
- CoRe 在 CoRe-Bench 上显著优于所有基线：平均 overall accuracy 达 22.2%（基线最高 6.6%），partial accuracy 达 54.0%（基线最高 25.8%），提升 28.2 点。

- 泛化性验证：CoRe 在 OOD 二图像基准上亦优于基线。
- 消融证实 TriSR 各组件均独立贡献，去除任意一个都会导致性能下降，其中三元组一致性（R_tc）贡献最大。

- 通用基准评估：CoRe 在 9 个基准中的 5 个上达到开源最佳，且未出现显著灾难性遗忘。
