## 摘要
多模态大语言模型（MLLMs）取得了显著进展，但在需要多步感知和逻辑推导的复杂视觉推理任务上仍面临挑战。显式视觉生成计算成本高昂，而现有潜空间方法往往依赖外部专家或缺乏严谨的认知逻辑。本文提出ProLaViT（渐进式潜视觉思考）框架，使MLLMs能够在连续潜空间中进行结构化视觉推导。与依赖异构外部模型的工作不同，ProLaViT利用内生自蒸馏机制，使用模型自身的视觉编码器监督潜思考。为此，我们构建了可扩展的程序化合成流水线，使模型能够在无需推理时工具的情况下内化算法精度。我们设计了两种推理范式：（1）用于空间任务的由粗到细因果链，引导注意力从全局语境转向局部目标；（2）用于逻辑任务的辩证推理链，引入反事实思考进行验证。此外，我们提出距离加权多样性损失，施加拓扑感知约束，通过强制语义区分性防止特征退化。大量实验表明，ProLaViT在面向视觉的基准测试中优于基线，以高效率实现了卓越的准确性和可解释性。

## Abstract
Multimodal Large Language Models (MLLMs) have achieved remarkable progress but still struggle with complex visual reasoning tasks requiring multi-step perception and logical deduction. While explicit visual generation incurs prohibitive computational costs, existing latent approaches often rely on external experts or lack rigorous cognitive logic. In this paper, we introduce ProLaViT (Progressive Latent Visual Thought), a framework empowering MLLMs to perform structured visual derivation in the continuous latent space. Unlike works dependent on heterogeneous external models, ProLaViT leverages an endogenous self-distillation mechanism, utilizing the model's own visual encoder to supervise latent thoughts. To facilitate this, we construct a scalable programmatic synthesis pipeline enabling the model to internalize algorithmic precision without inference time tools. We design two reasoning paradigms: (1) Coarse-to-Fine Causal Chain for spatial tasks, guiding attention from global context to local targets. (2) Dialectical Reasoning Chain for logical tasks, incorporating counter-factual thinking for verification. Furthermore, we propose a Distance-Weighted Diversity Loss to impose topology-aware constraints, preventing feature degeneration by enforcing semantic distinctiveness. Extensive experiments demonstrate that ProLaViT outperforms baselines on vision-centric benchmarks, achieving superior accuracy and interpretability with high efficiency.

## 2. 论文提出的方法论

### 核心思想
- 提出 **ProLaViT**（Progressive Latent Visual Thought）框架，将复杂视觉任务分解为一系列结构化的潜思考步骤（latent thoughts），进行渐进式推导。
- 设计两种推理范式：
  - **Coarse-to-Fine Causal Chain**（由粗到细因果链）：用于空间任务，步骤为 Locate → Focus → Isolate，逐步缩小注意力范围。
  - **Dialectical Reasoning Chain**（辩证推理链）：用于逻辑任务（如拼图），采用 Hypothesize → Critique → Verify 循环，引入反事实思考以验证假设。

### 关键技术细节
1. **渐进式潜视觉思考模块**：
   - 每个推理步骤用一组可学习 token（例如 4 个 <|vit_pad_latent|>）表示，共 K 步（K=4），总 token 数为 K×N_tokens。
   - 在 LLM 的输入序列中插入这些潜思考 token，通过自关注学习中间视觉状态。
2. **内生自蒸馏机制（Endogenous Self-Distillation）**：
   - 使用模型自身冻结的视觉编码器（Qwen2.5-VL 的 ViT）作为教师，为每个推理步骤生成目标视觉特征。
   - 通过程序化合成流水线，自动生成 K 个辅助图像（如裁剪、边界框、分割等），并提取其特征 F_teacher^k。
   - 利用单个跨注意力层将 LLM 生成的潜思考 token 表示与教师特征对齐，优化 L_distill = MSE(ˆF_k, F_teacher^k)。
3. **距离加权多样性损失（Distance-Weighted Diversity Loss）**：
   - 解决“潜崩塌”问题（后续步骤表示趋于相同）。
   - 基于 margin 的损失：L_div = 1/|P| Σ_(i,j) w_ij · max(0, cos(g_i, g_j) - τ)，其中 g_i 是第 i 步 token 的质心，τ 为可学习 margin，w_ij 由因果距离加权：w_ij = ε + (1-ε)·(d(i,j)/d_max)^α，确保相邻步骤保持一定相似性，而远距离步骤更区分。
4. **多阶段课程训练策略**：
   - Phase 1：冻结视觉编码器和 LLM 主干，只训练潜思考 token 的嵌入（锚点初始化）。
   - Phase 2：激活自蒸馏损失（L_distill），聚焦特征重建。
   - Phase 3：引入多样性损失（L_div），形成结构化推理链。
   - Phase 4：联合 VQA 数据训练，防止遗忘。

### 公式（文字说明）
- 潜思考 token 由 H_latent = LLM(E_input ⊕ E_thought) 生成。
- 自蒸馏：先投影 H 为 H_proj，然后计算跨注意力 ˆF = CrossAttn(Q_expand, H_proj, H_proj)，再分 K 段与教师特征计算 MSE。
- 多样性损失：计算各步质心 g_k 之间的余弦相似度，加权后仅惩罚超过阈值 τ 的相似度。
